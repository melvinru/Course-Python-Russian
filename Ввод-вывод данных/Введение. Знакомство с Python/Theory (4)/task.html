Среда разработки VS Code

Команды для интерпретатора можно писать в обычном текстовом редакторе (например, в "Блокноте"). Но чаще для этого пользуются специальной программой, которая называется средой разработки (англ. IDE, Integrated Development Environment).

Среда разработки — тоже текстовый редактор, но с дополнительными возможностями. Например, она умеет сама находить на компьютере программу-интерпретатор и запускать программу одной кнопкой. Среда разработки, кроме того, форматирует написанный вами код, чтобы его удобно было читать, а иногда даже подсказывает, где вы допустили ошибку.

На текущий момент очень популярной и удобной IDE является VS Code. Рекомендуем использовать её для решения задач в курсе и вообще в жизни.

В VS Code есть много расширений, которые позволяют настроить среду разработки под свои нужды. Есть расширения для работы с разными языками, сборки проектов, автоматического форматирования кода и многого другого. Это надёжная IDE, которую легко использовать.

Шаг 1. Скачиваем установщик VS Code

  1. Открываем окно браузера и переходим по ссылке
  2. Кликаем по синий кнопке Download for Windows (или кликаем по кнопке справа для выбора дополнительных параметров)

          

 Шаг 2. Устанавливаем VS Code

  1. Запускаем установщик программы
  2. Следуем указаниям установщика, нажимая кнопку Next
  3. Через несколько секунд на нашем компьютере будет установлена IDE VS Code

Шаг 3. Запускаем VS Code 

При открытии VS Code мы увидим экран приветствия

Тут же мы можем соответственно:

  • создать новый файл
  • открыть уже существующий
  • открыть папку с файлами (удобно при работе над проектами с несколькими файлами)

Шаг 4. Создаём первый проект на Python

Кликаем по кнопке New File, чтобы создать новый файл.

Вверху экрана появляется новое окно. Выбираем поле Text File Built-In.

 

 

 

 

У нас создаётся файл. После этого нажимаем на Select a language.                                

В открывшемся окне вводим python. И кликам по найденному языку Python. Теперь IDE будет понимать, что мы работаем с файлом формата py.

Однако для работы с Python файлами нам нужно будет установить расширение. Для этого переходим в раздел Extensions (расширения).

Ищем расширение для Python. Обычно оно первое в списке, это официальное расширения для python файлов от Microsoft. Скачиваем через кнопку Install (возможно, после установки придётся перезагрузить IDE для полноценной работы).

После установки расширения возвращаемся в наш файл с программой (пока ещё пустой). С помощью сочетания клавиш Ctrl + S сохраняем его у себя на диске. Можете создать отдельную папку на компьютере для своих Python проектов. Придумываем имя для своего файла и жмём кнопку Сохранить.

Далее нам надо проверить, что всё работает. Напишем простой код для проверки: print("Hello world").

Чтобы запустить программу, нажимаем на треугольник в правом верхнем углу. Также запуск программы возможно осуществить через Ctrl + F5 или просто F5.

Если мы сделали всё правильно, у нас появится окно терминала с выведенным текстом "Hello world".

Если у вас что-то не получилось, пишите в комментариях, разберёмся.     

Шаг 5. Некоторые рекомендации по работе с VS Code

Некоторые ошибки в программе будут прописаны в разделе PROBLEMS (в следующем случае мы специально удалили закрывающую скобку, что является синтаксической ошибкой в Python). Такое возможно благодаря плагину Pylance, который был автоматически установлен вместе с расширением Python.

В разделе Settings (настройки) можно включить автосохранение программы. Благодаря этому вам не придётся самим каждый раз нажимать Ctrl + S. Программа будет автоматически сохраняться спустя некоторое время. Для этого с помощью сочетания клавиш Ctrl + , переходим в Settings (также в настройки можно перейти через File -> Preferences -> Settings). Печатаем auto save и находим нужную настройку.

В открывшемся окне выбираем пункт afterDelay.

Автоформатирование программы можно включить также в разделе Settings​​​. Это очень удобная и полезная функция. Ваш файл будет автоматически форматироваться по стандартам PEP8 (официальный стандарт написания кода на языке Python). Для этого печатаем в настройках format on save и находим нужную настройку.

Ставим галочку.

При нажатии сочетания клавиш Ctrl + S у нас выскочит окошко, что IDE не видит форматер. Нажмём клавишу Yes, чтобы установился автоматически autopep8 форматер (также можете выбрать другие форматеры, предложены black и yapf).

Если же у вас такое окно не появилось, придётся установить форматер вручную. Это сделать можно через командную строку. Чтобы попасть в командную строку, нужно использовать клавиш Win + R, в появившемся окне "Выполнить" ввести команду cmd и нажать Enter. После этого скачаем форматер следующей командой: pip install autopep8.

Теперь наша программа будет форматироваться каждый раз после нажатия сочетания Ctrl + S (возможно, придётся перезапустить программу VS Code для работы автоформатера). Учитывайте, что программа будет форматироваться только при отсутствии синтаксических ошибок. Рекомендуем использовать следующий код для проверки (не вдавайтесь в подробности языка, пример только для демонстрации):

def get_sum(a,b)->int:
    return a+b
print(get_sum(5, 7))

При автоформатировании этот небрежный код.

Превращается в приятный, читабельный код (читабельность играет большую роль в программировании).                    

Стоит сказать, что иногда не получается установить автоформатер или он работает некорректно. В этом случае рекомендуется:

  1. Попробовать сделать всё с начала по шагам, возможно, сделали что-то не так;
  2. Установить форматер black вместо autopep8, т.к. autopep8 иногда работает некорректно;
  3. Погуглить (иногда бывают уникальные проблемы, которые решаются только так).

В любом случае установка форматера не является обязательной, больше носит рекомендательный характер. Кодить вы можете и без этого, тем более на первых шагах.

Изменить тему оформления в VS Code также возможно (если хотите, чтобы всё было красиво и приятно для глаз). Ищем в настройках color theme. И в этой настройке выбираем ту тему, которая вам понравится больше. Темы менять также можно через шестеренку в левом нижнем углу  -> Themes -> Color Theme. И дальше клавишами вверх-вниз листать (тема будет меняться на ходу, так что можно будет посмотреть все разу).

Изменить размер иконок в самой VS Code можно с помощью Ctrl + (увеличение) и Ctrl - (уменьшение).

В VS Code есть ещё очень много интересного. Можете поискать сами какие-нибудь классные расширения или удобные настройки.
