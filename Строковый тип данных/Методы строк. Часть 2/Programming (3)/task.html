Количество цифр

На вход программе подается строка текста. Напишите программу, которая подсчитывает количество цифр в данной строке.

Формат входных данных 
На вход программе подается строка текста.

Формат выходных данных
Программа должна вывести количество цифр в данной строке.

Тестовые данные 🟢
    Номер теста         Входные данные         Выходные данные    
1 nezabud dl-6 1
2 l33t 3301 6
3 1029384756aaa1234567890 20
4 jehfkjshfkjsdk90909090 8

Sample Input:
nezabud dl-6

Sample Output:
1


Sample Input:
l33t 3301

Sample Output:
6
