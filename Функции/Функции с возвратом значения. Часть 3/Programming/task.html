Середина отрезка

Напишите функцию get_middle_point(x1, y1, x2, y2), которая принимает в качестве аргументов координаты концов отрезка $(x_1; \, y_1)$ и $(x_2; \, y_2)$ и возвращает координаты точки являющейся серединой данного отрезка.

Примечание 1. Координаты середины отрезка с концами в точках $(x_1; y_1)$ и $(x_2; y_2)$ вычисляются по формуле:\[\left(\dfrac{x_1+x_2}{2}; \, \dfrac{y_1+y_2}{2}\right)\]

Примечание 2. Следующий программный код:

print(get_middle_point(0, 0, 10, 0))
print(get_middle_point(1, 5, 8, 3))

должен выводить:

5.0 0.0
4.5 4.0
